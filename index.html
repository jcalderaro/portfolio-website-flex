The Lambda Challenges

Technology is ever-changing. A fast-paced landscape that the students train for here at Lambda School.
In a web designer's career, it's important to have endurance. But you also need efficiency, speed, and adapatability.
After all, the hardest challenges come at you fast. Get ready to Sprint.

Sprint 1 - User Interface and Responsive Web Design

This challenge will cover all of the objectives from the sprint. You will be creating a fully responsive website from several design files. Each design file comes with a set width. Those widths will be used by you to compare your work against the design.

Sprint 2 - Advanced CSS and Preprocessors

This challenge applies the concepts and techniques of advanced CSS using Responsive Design and Preprocessing. During this Sprint, the usage of the viewport meta tag, media queries, setting up a preprocessor, and advanced use of preprocessing techniques will be demonstrated.

Sprint 3 - JavaScript Fundamentals

This Sprint explored JavaScript Fundamentals. During this Sprint, you studied variables, functions, object literals, arrays, this keyword, prototypes, and class syntax. In your challenge this week, you will demonstrate proficiency by completing a survey of JavaScript problems.
